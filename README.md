# 📘 База знаний IT

Добро пожаловать в мою открытую базу знаний по IT: теория, практические решения, заметки и всё, что может быть полезно в работе или при изучении технологий.

База построена на [Quartz](https://github.com/jackyzha0/quartz) и публикуется через GitHub Pages: **https://knowledge.alexandrshapkin.ru/**

---

## Что внутри

- Теоретические заметки по разным направлениям IT.  
- Практические подходы и решения, которые я применял на работе или учебе.  
- Материалы для переиспользования и дальнейшего изучения.

Все файлы в формате Markdown, их можно открыть напрямую в [Obsidian](https://obsidian.md) или любой другой Markdown-редактор.

---

## Зачем это

- Сохранять и структурировать знания.  
- Обеспечивать быстрый доступ к проверенным решениям.  
- Делиться наработками с другими и поддерживать рост базы совместно.

---

## 📂 Структура

Все заметки находятся в папке [`/content`](./content).  
Структура может меняться по мере роста базы, поэтому удобнее ориентироваться по папкам и тегам.

---

## 🤝 Как внести вклад

Если хотите добавить или исправить заметку:  

1. Форкните репозиторий.  
2. Создайте ветку для своих изменений.  
3. Отредактируйте файлы в Obsidian или напрямую в Markdown.  
4. Сделайте Pull Request в ветку `v4`.  

Подробная инструкция для контрибьюторов есть в [`CONTRIBUTING.md`](./CONTRIBUTING.md).
