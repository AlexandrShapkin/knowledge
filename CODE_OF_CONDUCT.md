# Кодекс поведения для базы знаний

## 1. Цель

Этот репозиторий создан для совместного развития базы знаний. Мы хотим, чтобы все участники — независимо от опыта и происхождения — могли вносить свой вклад в дружелюбной и уважительной атмосфере.

## 2. Рекомендованное поведение

Участникам рекомендуется:

- Общаться вежливо и уважительно в Pull Request, issue и обсуждениях.  
- Сотрудничать конструктивно и помогать другим участникам.  
- Избегать оскорблений, дискриминации и уничижительных комментариев.  
- Учитывать, что ваши изменения могут видеть и использовать другие.

## 3. Область применения

Этот Кодекс касается всех взаимодействий в репозитории, включая:

- Pull Request  
- Issues  
- Обсуждения  
- Комментарии  
- Любое общение, связанное с репозиторием

## 4. Лицензия и атрибуция
