`%CPU` — доля процессорного времени, использованная процессом за интервал обновления.

Кратко:

- Рассчитывается как отношение CPU-времени процесса к прошедшему времени между обновлениями.
- Основан на накопленных счётчиках user time и system time.
- Отражает текущую вычислительную активность процесса.

Особенности:

- В многопроцессорной системе может превышать 100% (если процесс использует несколько ядер).
- Значение зависит от периода обновления `top`/`htop`.
- Для короткоживущих процессов может резко изменяться.

Не показывает общее время жизни процесса, а характеризует интенсивность использования CPU в текущий момент. Используется для выявления CPU-bound задач и анализа распределения нагрузки.
