`buffers` — объём памяти, используемой ядром для буферизации операций с блочными устройствами.

Кратко:

- Содержит служебные структуры и метаданные файловых систем (например, информацию о блоках, [inode](../../../../inode.md)).
- Используется при работе с диском для ускорения операций ввода-вывода.
- Может быть освобождён ядром при необходимости.

Обычно составляет небольшую часть RAM и редко является причиной нехватки памяти.
