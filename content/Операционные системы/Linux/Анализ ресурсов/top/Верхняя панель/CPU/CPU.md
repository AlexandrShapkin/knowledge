 Метрики CPU в [top](../../top.md) / [htop](../../htop.md) показывают распределение времени процессора по типам активности. Значения выражаются в процентах от общего времени CPU.

- [user (us)](user%20%28us%29.md) — время выполнения пользовательского кода (обычные процессы вне ядра).
- [system (sy)](system%20%28sy%29.md) — время работы ядра (системные вызовы, обслуживание процессов, драйверы).
- [nice (ni)](nice%20%28ni%29.md) — время выполнения процессов с изменённым приоритетом (nice ≠ 0).
- [idle (id)](idle%20%28id%29.md) — время простоя CPU, когда нет задач к выполнению.
- [iowait (wa)](iowait%20%28wa%29.md) — время ожидания завершения операций ввода-вывода (диск, сеть).
- [hardware interrupt (hi)](hardware%20interrupt%20%28hi%29.md) — обработка аппаратных прерываний (IRQ от устройств).
- [software interrupt (si)](software%20interrupt%20%28si%29.md) — обработка программных прерываний (softirq, сетевые стеки и др.).
- [steal (st)](steal%20%28st%29.md) — время, «отнятое» гипервизором в виртуальной машине (CPU был доступен хосту, но не гостю).
- [guest](guest.md) — время, затраченное на выполнение виртуальных машин внутри текущей системы (если используется KVM и т.п.).
- [загрузка по каждому ядру](Загрузка%20по%20каждому%20ядру.md) — те же показатели, но отдельно для каждого логического CPU, что позволяет выявлять неравномерность распределения нагрузки.
