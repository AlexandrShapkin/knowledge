---
title: Что такое K3s?
---
[K3s](https://k3s.io/) — это полностью совместимая дистрибуция Kubernetes с рядом улучшений:
- Поставляется как единый бинарный файл или минимальный контейнерный образ.
- Лёгкое хранилище данных на базе sqlite3 используется по умолчанию. Также поддерживаются etcd3, MySQL и Postgres.
- Обёрнут в простой лаунчер, который берёт на себя значительную часть сложности работы с TLS и настройками.
- Безопасен «из коробки» благодаря адекватным настройкам по умолчанию для лёгких сред.
- Работа всех компонентов control plane Kubernetes инкапсулирована в один бинарный файл и процесс, что позволяет K3s автоматизировать и управлять сложными операциями кластера, например распределением сертификатов.
- Внешние зависимости сведены к минимуму; требуются лишь современное ядро и монтирование cgroup.
- Включает необходимые зависимости для лёгкого создания кластера по принципу «всё включено»:
    - containerd / cri-dockerd (контейнерный runtime, CRI)
    - Flannel (CNI для сетей контейнеров)
    - CoreDNS (кластерный DNS)
    - Traefik (Ingress-контроллер)
    - ServiceLB (контроллер балансировки нагрузки)
    - Kube-router (контроллер сетевых политик)
    - Local-path-provisioner (контроллер постоянных томов)
    - Spegel (распределённое зеркало реестра контейнерных образов)
    - Системные утилиты (iptables, socat и др.)
