В системе DNS используются **ресурсные записи (DNS records)** — это инструкции, которые определяют, как [доменное имя](DNS-домен.md) связывается с IP-адресами и другими данными.
## Основные типы DNS-записей
- **[A](A-запись.md)** — связывает доменное имя с IPv4-адресом.
- **[AAAA](AAAA-запись.md)** — связывает доменное имя с IPv6-адресом.
- **[CNAME](CNAME-запись.md)** — задаёт псевдоним: одно доменное имя указывает на другое.
- **MX** — определяет почтовые серверы для домена.
- **NS** — указывает DNS-серверы, авторитетные для домена.
- **TXT** — хранит произвольный текст (обычно для верификации или SPF/DKIM).
- **SRV** — задаёт расположение сервиса (порт, хост).
- **PTR** — используется для обратного DNS (IP → доменное имя).
- **SOA** — содержит информацию о зоне: основной сервер, контакты администратора, время кэширования.

**Итог:**  
DNS-записи определяют, как имя домена должно интерпретироваться: какой у него IP-адрес, какой сервер обрабатывает почту, какие NS считаются авторитетными и т.д. Это основа работы DNS.
