Цель: убедиться, что ключевые сервисы доступны и работают.

**Сетевые соединения**

- Проверить, слушают ли нужные порты.  

Команда:
```bash
ss -tlnp
```

**Базовая связность**

- Проверить доступность DNS и внешних ресурсов.  

Команды:
```bash
ping -c 2 ya.ru ping -c 2 <IP сервера>
```
