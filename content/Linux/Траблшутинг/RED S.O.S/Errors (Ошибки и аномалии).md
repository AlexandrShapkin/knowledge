Цель: быстро найти критические сообщения и сетевые проблемы.

**Системные журналы**

- Проверка ошибок последних нескольких минут.  

Команда:
```bash
journalctl --since "5 minutes ago" -p err..emerg
```

Ищем:

- OOM-killer сообщения
- аппаратные сбои
- проблемы с файловой системой

**Сетевые ошибки**

- Смотреть _errors_, _dropped_ в статистике интерфейсов.  

Команда:
```bash
ip -s link
```
