`TIME+` — суммарное процессорное время, потреблённое процессом с момента запуска.

Кратко:

- Представляет сумму user time и system time.
- Отображается в формате `часы:минуты:секунды.сотые`.
- Основано на накопленных счётчиках ядра.

Особенности:

- Учитывает только фактическое выполнение на CPU.
- Не включает периоды ожидания (sleep, I/O).
- Растёт монотонно до завершения процесса.

Не отражает текущую нагрузку, а показывает общий объём выполненной процессором работы. Используется для оценки вычислительной «тяжести» процесса за всё время его жизни.
