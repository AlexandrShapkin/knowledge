`cumulative CPU time` — накопленное процессорное время процесса с момента его запуска.

Кратко:

- Представляет сумму времени в user mode и kernel mode.
- Учитывается только фактическое выполнение на CPU.
- Основано на счётчиках времени в ядре (`utime` + `stime`).

Особенности:

- Не включает периоды ожидания (sleep, I/O, блокировки).
- Растёт монотонно до завершения процесса.
- Используется как база для расчёта `%CPU`.

Не характеризует текущую активность, а отражает общий объём вычислительной работы, выполненной процессом за всё время его существования.
