`last used CPU` — номер процессора или ядра, на котором процесс выполнялся в последний раз.

Кратко:

- Определяется планировщиком ядра при последнем выделении процессора для выполнения задачи.
- Показывает, на каком CPU процесс был активен недавно.
- Используется для анализа распределения нагрузки и affinity процессов.

Особенности:

- Не гарантирует, что процесс продолжит выполняться на том же ядре.
- Полезно для диагностики локализации вычислений и выявления горячих ядер.
- Не отражает ни суммарное использование CPU, ни текущее состояние процесса.
