`USER` — строковое имя пользователя, от имени которого выполняется процесс.

Кратко:

- Является человекочитаемым представлением [UID](UID.md).
- Получается через систему учётных записей (NSS, `/etc/passwd` и др.).
- Используется для удобства анализа в инструментах мониторинга.

Не влияет на права доступа напрямую — контроль осуществляется по числовому `UID`.
Применяется для быстрой идентификации владельца процесса при диагностике.
