 `np.maximum(X, 0)` — это **[[Векторизация как концепция|векторная]] поэлементная операция**, которая для каждого элемента [[ndarray|массива]] `X` выбирает **максимум между значением и 0**.

Формально:

```
result[i] = max(X[i], 0)
```

Пример:

```python
import numpy as np

X = np.array([[ 1, -2,  3],
              [-4,  5, -6]])

Y = np.maximum(X, 0)
print(Y)
```

Результат:

```
[[1 0 3]
 [0 5 0]]
```

Что важно:

– операция выполняется **[[Почему циклы хуже в NumPy|без циклов]]**, полностью векторизованно  
– работает по правилам **[[Broadcasting идея и назначение|broadcasting]]** (второй аргумент — скаляр)  
– возвращает новый массив  
– по смыслу эквивалентно условию `X[X < 0] = 0`, но **без индексации и масок**

Типичное применение:

– обрезка отрицательных значений (clipping снизу)  
– ReLU в машинном обучении  
– очистка шумов и артефактов

Сравнение с булевой маской:

```python
Y = np.maximum(X, 0)
```

vs

```python
Y = X.copy()
Y[Y < 0] = 0
```

`np.maximum`:  
– короче  
– обычно быстрее  
– выражает намерение декларативно

Коротко:  
`np.maximum(X, 0)` — это векторный способ заменить все отрицательные значения в массиве на нули без циклов и явных условий.
