 `X > 0` в NumPy — это **[[Векторизация как концепция|векторное]] сравнение**, результатом которого является **[[Булевы маски|булева маска]]**.

Что происходит:

– операция применяется **поэлементно**  
– каждый элемент массива сравнивается с `0`  
– результат — массив `bool` той же формы, что и `X`

Пример:

```python
import numpy as np

X = np.array([[1, -2, 0],
              [3, -4, 5]])

mask = X > 0
print(mask)
```

Результат:

```
[[ True False False]
 [ True False  True]]
```

Практическое использование:

1. **Фильтрация элементов**

```python
X[X > 0]
# [1 3 5]
```

2. **Условное изменение значений**

```python
X[X > 0] = 1
X[X <= 0] = 0
```

3. **Подсчёт элементов по условию**

```python
np.sum(X > 0)   # количество положительных элементов
```

4. **Комбинация условий**

```python
(X > 0) & (X < 10)
```

Важно помнить:

– `X > 0` не возвращает одно `True/False`, а **массив**  
– `and / or` не работают, только `& / |`  
– результат можно использовать напрямую как индекс

Коротко:  
`X > 0` — это векторное условие, которое превращает массив чисел в булеву маску для фильтрации и управления данными [[Почему циклы хуже в NumPy|без циклов]].
