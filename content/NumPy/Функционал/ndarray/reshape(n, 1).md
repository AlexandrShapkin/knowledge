 `reshape(n, 1)` в NumPy используется для **преобразования [[ndarray|массива]] в двумерный [[Матрица-строка и матрица-столбец|массив-столбец]]**.

Если у вас есть одномерный массив длины `n`:

```python
import numpy as np

a = np.array([1, 2, 3, 4])
print(a.shape)  # (4,)
```

Применение `reshape(n, 1)`:

```python
b = a.reshape(4, 1)
print(b)
# [[1]
#  [2]
#  [3]
#  [4]]
print(b.shape)  # (4, 1)
```

Особенности:

- Превращает 1D массив `(n,)` в 2D массив с одной колонкой `(n, 1)`
- Используется для **[[Broadcasting идея и назначение|broadcasting]] по столбцам** или для операций линейной алгебры, где нужна матрица-столбец
- Не копирует данные, если это возможно — создаётся view с изменёнными `shape` и `strides`

Пример с broadcasting:

```python
x = np.array([1, 2, 3])
y = np.array([10, 20])

x_col = x.reshape(3, 1)  # shape (3, 1)

# Поэлементное сложение с матрицей shape (2,)
result = x_col + y       # shape (3, 2)
print(result)
# [[11 21]
#  [12 22]
#  [13 23]]
```

Идея: `reshape(n, 1)` делает массив **двумерным столбцом**, чтобы его можно было согласовать с другими 2D массивами для операций без циклов.
