`np.zeros` создаёт [[ndarray|массив]] заданной [[Понятие формы массива (shape)|формы]], **заполненный нулями**. Это полезно для инициализации массивов перед вычислениями.

Синтаксис:

```python
np.zeros(shape, dtype=float, order='C')
```

- `shape` — форма массива, кортеж чисел (например `(2, 3)` для матрицы 2×3)
- `dtype` — тип данных элементов (по умолчанию `float64`)
- `order` — порядок хранения в памяти (`'C'` — построчно, `'F'` — постколонно)

Примеры:

```python
import numpy as np

# Одномерный массив из 5 нулей
a = np.zeros(5)
print(a)   # [0. 0. 0. 0. 0.]

# Матрица 2x3 из нулей
b = np.zeros((2, 3))
print(b)
# [[0. 0. 0.]
#  [0. 0. 0.]]

# Матрица 2x3 целого типа
c = np.zeros((2, 3), dtype=int)
print(c)
# [[0 0 0]
#  [0 0 0]]
```

Особенности:

- Возвращает **ndarray** с указанной формой
- Эффективно выделяет память, все элементы инициализированы нулями
- Полезно для инициализации результатов, буферов, счетчиков, матриц для линейной алгебры
