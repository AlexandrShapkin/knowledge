 Для любого [[ndarray|numpy.ndarray]] основные характеристики массива получают через свойства:

- **[[Понятие формы массива (shape)|shape]]** — кортеж размеров массива по каждой оси.
- **[[Размерность массива (ndim)|ndim]]** — количество измерений (осей).
- **[[Количество элементов (size)|size]]** — общее число элементов (произведение размеров в `shape`).

Примеры:

```python
import numpy as np

a = np.array([[1, 2, 3],
              [4, 5, 6]])

a.shape   # (2, 3) — 2 строки, 3 столбца
a.ndim    # 2      — двумерный массив
a.size    # 6      — всего 6 элементов
```

Для 1D массива:

```python
b = np.array([1, 2, 3, 4])
b.shape   # (4,)
b.ndim    # 1
b.size    # 4
```

Для 3D массива:

```python
c = np.zeros((2, 3, 4))
c.shape   # (2, 3, 4)
c.ndim    # 3
c.size    # 24
```

Эти свойства **не создают копий данных**, а только читают метаданные массива.
