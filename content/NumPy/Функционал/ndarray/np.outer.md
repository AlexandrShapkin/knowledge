`np.outer` — это функция NumPy для вычисления **внешнего произведения** двух одномерных [[ndarray|массивов]].

Суть: если есть векторы `a` и `b`, `np.outer(a, b)` создаёт [[Одномерный массив vs матрица|матрицу]], где каждый элемент равен произведению соответствующих элементов из `a` и `b`:

```
C[i, j] = a[i] * b[j]
```

Пример:

```python
import numpy as np

a = np.array([1, 2, 3])
b = np.array([10, 20])

C = np.outer(a, b)
print(C)
# [[ 10  20]
#  [ 20  40]
#  [ 30  60]]
print(C.shape)  # (3, 2)
```

Особенности:

- Работает только с одномерными массивами
- Результат всегда двумерный массив `(len(a), len(b))`
- Используется для создания матриц коэффициентов, сеток и при линейной алгебре (тензорные произведения в простом виде)

Пример применения:

- Генерация матрицы, где строки — копии вектора с умножением на элементы другого вектора
- В статистике или ML для построения комбинаций признаков

Коротко: `np.outer(a, b)` — это **поэлементное умножение каждого элемента первого вектора на каждый элемент второго**, результат — [[Одномерный массив vs матрица|матрица]].
