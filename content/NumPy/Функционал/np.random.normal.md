 `np.random.normal` — это функция NumPy для **генерации случайных чисел из [[Нормальное распределение|нормального (гауссового) распределения]]**.

Синтаксис:

```python
np.random.normal(loc=0.0, scale=1.0, size=None)
```

Параметры:
- `loc` — [[Параметры μ и σ|среднее значение μ]] (центр распределения)
- `scale` — [[Параметры μ и σ|стандартное отклонение σ]] (разброс)
- `size` — [[Понятие формы массива (shape)|форма]] возвращаемого массива (скаляр, 1D, 2D и т.д.)

Примеры:

1D массив:

```python
import numpy as np

samples = np.random.normal(loc=0, scale=1, size=5)
print(samples)
# Например: [ 0.23 -1.14  0.78  0.12 -0.56 ]
```

2D массив:

```python
samples_2d = np.random.normal(loc=10, scale=2, size=(3,4))
print(samples_2d)
```

Особенности:

- Все элементы **независимы**
- Среднее ≈ `loc`, стандартное отклонение ≈ `scale` при большой выборке
- Можно использовать для моделирования шума, симуляций и тестирования алгоритмов

Применение в статистике и ML:

- Генерация синтетических данных
- Добавление нормального шума к сигналам
- Проверка алгоритмов на выборках с известным распределением

Коротко:  
**`np.random.normal` создаёт [[ndarray|массив]] случайных чисел, распределённых нормально с заданными средним и стандартным отклонением.**
