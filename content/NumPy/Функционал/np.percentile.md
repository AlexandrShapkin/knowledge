`np.percentile` — это функция NumPy для вычисления **[[Квантиль как устойчивая мера|квантили]] в процентах**, то есть значения, ниже которого находится заданная доля данных. Она особенно удобна для анализа многократных экспериментов и экстремальных статистик.

Синтаксис:

```python
np.percentile(a, q, axis=None)
```

- `a` — [[ndarray|массив данных]]
- `q` — процентиль (число от 0 до 100)
- `axis` — [[Понятие осей (axis)|ось]], по которой вычислять (по умолчанию по всему массиву)

Примеры:

```python
import numpy as np

data = np.array([1, 2, 3, 4, 5])

median = np.percentile(data, 50)    # медиана
q25 = np.percentile(data, 25)       # первый квартиль
q75 = np.percentile(data, 75)       # третий квартиль
```

Особенности:

- `np.percentile` **устойчива к выбросам**, в отличие от [[Вычисление среднего|среднего]]
- Можно вычислять **экстремальные процентили**, например 95-й или 99-й процентиль, чтобы оценить максимум без влияния единичных выбросов
- Работает с многомерными массивами при указании `axis`, что удобно для [[Многократные эксперименты|многократных экспериментов]]

Пример для многократных экспериментов с матрицей Вигнера:

```python
import numpy as np

n = 100
experiments = 1000
X_all = np.random.normal(0, 1, size=(experiments, n, n))
W_all = (X_all + X_all.transpose(0,2,1)) / np.sqrt(2*n)
maxima = W_all.max(axis=(1,2))

p95 = np.percentile(maxima, 95)  # 95-й процентиль максимума
```

Интерпретация:

- `p95` показывает, что 95% максимальных значений меньше этого числа
- Используется для оценки экстремальных случаев без влияния единичных редких пиков

Коротко:  
**`np.percentile` вычисляет значение, ниже которого находится заданная доля данных, и служит устойчивой мерой положения или экстремальных значений в массиве.**
