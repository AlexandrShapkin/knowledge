 `np.random.poisson` — это функция NumPy для **генерации случайных чисел из [[Распределение Пуассона|распределения Пуассона]]**.

Синтаксис:

```python
np.random.poisson(lam=1.0, size=None)
```

Параметры:
- `lam` — [[Параметр λ|параметр λ]], среднее число событий за интервал
- `size` — [[Понятие формы массива (shape)|форма]] возвращаемого массива (скаляр, 1D, 2D и т.д.)

Примеры:

1D массив:

```python
import numpy as np

samples = np.random.poisson(lam=3, size=10)
print(samples)
# Например: [2 5 3 1 4 3 3 2 4 0]
```

2D массив:

```python
samples_2d = np.random.poisson(lam=5, size=(3,4))
print(samples_2d)
```

Особенности:

- Все элементы **независимы**
- Среднее и дисперсия ≈ λ при большой выборке
- Используется для моделирования **редких событий за фиксированный интервал** (звонки, клики, ошибки)

Применение:

- Симуляции процессов с редкими событиями
- Тестирование алгоритмов для дискретных данных
- Моделирование потоков событий в очередях, сетях и сервисах

Коротко:  
**`np.random.poisson` создаёт [[ndarray|массив]] случайных целых чисел, распределённых по Пуассону с заданным средним λ.**
