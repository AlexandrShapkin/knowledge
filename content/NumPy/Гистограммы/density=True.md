`density=True` в [[plt.hist|plt.hist]] включает **нормализацию [[Что такое гистограмма|гистограммы]]**, при которой она интерпретируется как **оценка плотности распределения**, а не как счётчик элементов.

Что это означает:

– высота столбцов — не количество элементов  
– **площадь под всей гистограммой равна 1**  
– гистограмму можно сравнивать с теоретической плотностью распределения

Формально для каждого бина:

```
height = count / (N * bin_width)
```

где  
– `count` — число элементов в бине  
– `N` — размер выборки  
– `bin_width` — ширина интервала

Пример:

```python
import numpy as np
import matplotlib.pyplot as plt

data = np.random.normal(0, 1, 1000)

plt.hist(data, bins=30, density=True)
plt.show()
```

Если добавить теоретическую плотность:

```python
x = np.linspace(-4, 4, 200)
pdf = (1 / np.sqrt(2*np.pi)) * np.exp(-0.5 * x**2)

plt.hist(data, bins=30, density=True)
plt.plot(x, pdf)
plt.show()
```

Когда использовать `density=True`:

– при сравнении выборки с теоретическим распределением  
– при сравнении гистограмм с разным размером выборки  
– когда интересует форма распределения, а не абсолютные частоты

Когда не нужно:

– если важно реальное количество наблюдений в интервалах  
– для отчётов, где интерпретируется именно число элементов

Коротко:  
**`density=True` превращает гистограмму из счётчика наблюдений в нормированную оценку плотности распределения с площадью 1.**
