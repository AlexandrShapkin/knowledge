 `plt.hist` — это функция Matplotlib для **построения [[Что такое гистограмма|гистограммы]]**, то есть визуализации эмпирического распределения данных.

Базовый вызов:

```python
import matplotlib.pyplot as plt

plt.hist(data)
plt.show()
```

Что делает `plt.hist`:

– разбивает диапазон значений на интервалы ([[Влияние числа bins|bins]])  
– считает, сколько элементов попало в каждый интервал  
– рисует столбцы с высотой, пропорциональной частоте

Основные параметры:

`bins` — число интервалов или их явные границы

```python
plt.hist(data, bins=20)
```

`density` — нормализация

```python
plt.hist(data, density=True)
```

Если `True`, площадь под гистограммой равна 1 (приближение плотности распределения).

`range` — диапазон значений

```python
plt.hist(data, range=(-3, 3))
```

`alpha` — прозрачность (удобно для наложения гистограмм)

```python
plt.hist(data, alpha=0.7)
```

`histtype` — тип отображения (`bar`, `step`, `stepfilled`)

Пример с нормальным распределением:

```python
import numpy as np
import matplotlib.pyplot as plt

data = np.random.normal(0, 1, 1000)

plt.hist(data, bins=30, density=True)
plt.show()
```

Интерпретация результата:

– форма столбцов отражает **эмпирическое распределение**  
– увеличение `bins` повышает детализацию, но добавляет шум  
– `density=True` позволяет сравнивать с теоретическими плотностями

Коротко:  
**`plt.hist` — основной инструмент для визуального анализа распределения данных через гистограммы.**
