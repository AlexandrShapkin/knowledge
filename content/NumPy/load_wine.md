 `load_wine` — это функция из `sklearn.datasets`, которая загружает **Wine dataset**, популярный [[Что такое датасет|датасет]] для задач классификации и анализа [[Многомерные данные|многомерных данных]].

Основные характеристики:

1. **Размерность и структура**

- 178 объектов (вино)
- 13 признаков на объект
- 3 класса вина

2. **Признаки**  
    Примеры: содержание алкоголя, кислотность, цветность, магний и др.
3. **Целевая переменная**

- Метка класса (`target`) — целое число `0`, `1` или `2`

2. **Использование в NumPy и Scikit-learn**

```python
from sklearn.datasets import load_wine

data = load_wine()
X = data.data        # признаки (178, 13)
y = data.target      # метки классов (178,)
feature_names = data.feature_names  # названия признаков
target_names = data.target_names    # названия классов
```

5. **Типичный рабочий процесс**

- `X` → [[Зачем масштабировать признаки|масштабирование]], [[Стандартизация (Z-score)|стандартизация]]
- `y` → целевая переменная для классификации
- Можно визуализировать 2–3 признака, использовать для обучения моделей (логистическая регрессия, SVM, kNN)

Коротко:  
**`load_wine` загружает готовый набор данных о винах с признаками и метками классов, удобный для обучения и экспериментов с классификацией.**
